# Alfred-Personal Data Butler

![banner](https://s1.ax1x.com/2020/04/18/JmVm8I.png)


Alfred是一款私有云盘，通过简单的搭建可以使个人轻松的使用。

Alfred支持各种文件类型，支持大文件上传，支持秒传、分块上传、断点续传、异步队列上传。同时，在数据方面支持将文件上传至多个云端，如阿里云。同时，用户可搭建自己的ceph集群用于存储数据。
